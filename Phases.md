# PetPro Development Phases

## Phase 1: Pre-Testing (CURRENT)
- Building the core app from scratch
- Getting all features working with fake data
- Steps 1-17 of the build order
- UI is functional but not polished
- Nicole decides when this phase is done

## Phase 2: Alpha
- Nicole uses PetPro daily at her own grooming business
- Real clients, real dogs, real bookings
- Use during the day, fix and add features every night
- Find bugs and missing features through daily real-world use
- Claude tells Nicole when it's ready to move to Beta

## Phase 3: Beta
- App is polished with better UI and more detail
- Pre-made service templates (5+ common services already set up)
- New groomers can edit the pre-made services instead of building from scratch
- Invite groomer friends to test for free
- Collect feedback and fix issues

## Phase 4: Public Launch
- Subscription payments live via Stripe
- Basic plan $40/month / Pro plan $70/month
- Marketing to solo groomers and small shops
- Full feature set working and tested

## Pre-Made Service Templates (for Beta/Launch)
- These will come pre-loaded so new groomers can just edit prices:
  - Full Groom by weight ranges (under 10lbs, 10-15, 15-25, 25+)
  - Bath & Brush by weight ranges
  - Puppy Introduction to Grooming (under 4 months)
  - Puppy Groom (4-8 months)
  - Face, Feet & Sanitary Trim
  - Common add-ons (dematting, handling, express, deshed, flea, teeth, nails)
- Groomers can edit any of these or add their own custom services
