## Session 1 Notes - April 14, 2026

### Claude Console vs Claude Chat App
- **Claude chat** (claude.ai or this desktop app) = where you talk to Claude
- **Claude Console** (console.anthropic.com) = developer dashboard for API keys and billing
- They are two different things, same company
- The Console is where you get the key that lets PetPro's code talk to Claude's brain
- API is pay-as-you-go (pennies per test during development)

### Pricing Plans for PetPro Customers
- Basic plan $40/month - calendar, profiles, automated texts, no voice or AI safety
- Pro plan $70/month - full Claude AI brain, voice commands, all features

### Key Reminders
- Go step by step, one at a time
- Never skip ahead in build order
- Ask questions rather than assuming
- API keys are like passwords - never share them
- Free tiers will carry us through most of early development
